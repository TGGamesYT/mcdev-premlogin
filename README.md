# mcdev-premlogin

An IntelliJ plugin for Minecraft development that allows pre-login with Microsoft and Cracked accounts directly in the IDE. Designed to work alongside the [Minecraft Dev Plugin](https://github.com/minecraft-dev/MinecraftDev).

## Features

- New settings menu "Minecraft Accounts" in settings.
- Add Microsoft accounts via OAuth login.
- Add Cracked accounts.
- Automatically updates Minecraft Client run configuration with the selected account.
- Handles Minecraft, Xbox Live, and XSTS authentication tokens.
- Supports custom `.idea/runConfigurations` directory.
